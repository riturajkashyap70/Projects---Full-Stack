# DefectTracker Pro – Bug Assignment & Monitoring System

## 📌 Project Overview

DefectTracker Pro is a full-stack bug tracking and monitoring system designed to manage software defects efficiently.
The system helps teams track bugs, assign them to developers, monitor progress, and maintain organized defect records in one centralized platform.

This project demonstrates practical implementation of **full-stack development concepts**, database management, and responsive frontend UI.

---

## 🚀 Features

* User Authentication (Login / Logout)
* Bug Reporting System
* Bug Assignment to Developers
* Bug Status Tracking (Open / In Progress / Resolved)
* Dashboard for Monitoring Defects
* Search and Filter Bugs
* Centralized defect management

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/riturajkashyap70/Projects---Full-Stack.git
```

### 2️⃣ Navigate to project folder

```
cd DefectTracker-Pro
```

### 3️⃣ Create virtual environment

```
python -m venv venv
```

### 4️⃣ Activate environment

Windows:

```
venv\Scripts\activate
```

### 5️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 6️⃣ Run migrations

```
python manage.py migrate
```

### 7️⃣ Start server

```
python manage.py runserver
```
